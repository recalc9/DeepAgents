## 使用 apply_patch 的规则
- 创建新文件时，描述完整文件内容，工具会自动生成 diff
- 修改现有文件时，只描述需要改动的部分，工具生成增量 patch
- 移动/重命名文件时，在 update 操作中指定 moveTo 目标路径
- 编辑前先用 file_read 查看当前内容，确保 patch 能正确应用
